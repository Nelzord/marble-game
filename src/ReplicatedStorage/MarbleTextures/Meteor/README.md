# Meteor Marble Texture

This folder contains texture assets for the Meteor marble.

## Suggested Texture Style
- Primary colors: Orange-red (#FF6400), yellow-orange, and dark red
- Pattern: Meteor/asteroid patterns with fire and space effects
- Material feel: Fiery space rock with glowing embers and cosmic textures
- Theme: Space/meteor vibes with fire and impact effects

## Texture File
Place a texture file named `MeteorTexture.txt` containing the Roblox asset ID for the marble texture.

## Rarity
1/75 - Rare marble with powerful teleport and meteor rain ability
