# Coin Marble Texture

This folder contains texture assets for the Coin marble.

## Suggested Texture Style
- Primary colors: Gold (#FFD700), yellow, and bronze
- Pattern: Coin patterns with dollar signs, coins, or treasure effects
- Material feel: Shiny gold with metallic reflections and coin textures
- Theme: Treasure/coin vibes with gold and money effects

## Texture File
Place a texture file named `CoinTexture.txt` containing the Roblox asset ID for the marble texture.

## Rarity
1/20000 - Extremely rare marble with passive coin-dropping ability
