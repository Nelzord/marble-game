# Volcano Marble Texture

This folder contains texture assets for the Volcano marble.

## Suggested Texture Style
- Primary colors: Red-orange (#FF4500), dark red, and black
- Pattern: Magma/lava cracks with glowing orange veins
- Material feel: Volcanic rock with molten lava flowing through cracks

## Texture File
Place a texture file named `VolcanoTexture.txt` containing the Roblox asset ID for the marble texture.
