# Leaf Marble Texture

Place Decal or Texture instances in this folder to customize the Leaf marble appearance.

## Recommended Design
- Use green, leafy patterns or autumn-colored leaves
- Forest green (34, 139, 34) is the base color
- Consider using leaf/nature themed textures

## Fallback
If no textures are placed here, the marble will use the fallback decal defined in InventorySystem.luau.
