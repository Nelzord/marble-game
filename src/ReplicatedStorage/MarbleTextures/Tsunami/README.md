# Tsunami Marble Texture

This folder contains texture assets for the Tsunami marble.

## Suggested Texture Style
- Primary colors: Ocean blue (#64C8FF), cyan, white foam, and teal
- Pattern: Wave patterns with sparkles and ocean foam effects
- Material feel: Shimmering ocean water with light reflections, sparkles, and wave textures
- Theme: Ocean/tsunami vibes with sparkling water effects

## Texture File
Place a texture file named `TsunamiTexture.txt` containing the Roblox asset ID for the marble texture.

## Rarity
1/100 - Rare marble with powerful wave-launching ability
