# Phoenix Marble Model

This folder should contain the custom model for the Phoenix marble.

## Model Requirements
- Must be a Roblox Model instance
- Should have a spherical base shape
- Recommended visual elements:
  - Yellow-orange and blue flame texture (using the texture ID: rbxassetid://124481451788068)
  - Glowing blue and yellow flame effects (using Neon material)
  - Blue and yellow point lights for magical glow effect
  - Optional: Small sparkle particles emanating from the marble

## Color Scheme
- Base: Yellow-orange (#FFC800) with blue (#64B4FF) accents
- Accents: Blue and yellow glowing flames
- Glow: Warm yellow and cool blue lights
