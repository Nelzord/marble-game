# Space Marble Model

This folder should contain the custom model for the Space marble.

## Model Requirements
- Must be a Roblox Model instance
- Should have a spherical base shape
- Recommended visual elements:
  - Space/cosmic texture (using the texture ID: rbxassetid://119305575673092)
  - Rainbow sparkle particles
  - Rainbow trail effects
  - Optional: Star particles or cosmic glow effects

## Color Scheme
- Base: Purple (#6432C8) with space/cosmic patterns
- Effects: Rainbow sparkles and trail
- Glow: Magenta/pink rainbow light
