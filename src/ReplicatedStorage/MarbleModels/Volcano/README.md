# Volcano Marble Model

This folder should contain the custom model for the Volcano marble.

## Model Requirements
- Must be a Roblox Model instance
- Should have a spherical base shape
- Recommended visual elements:
  - Dark volcanic rock texture
  - Glowing lava cracks/veins (using Neon material)
  - Orange/red point light for lava glow effect
  - Optional: Small smoke particles emanating from cracks

## Color Scheme
- Base: Dark gray/black volcanic rock
- Accents: Orange (#FF4500) and red (#FF0000) glowing veins
- Glow: Warm orange light
