# Hacker Marble Model

This folder should contain the custom model for the Hacker marble.

## Model Requirements
- Must be a Roblox Model instance
- Should have a spherical base shape
- Recommended visual elements:
  - Hacker/binary texture (using the texture ID: rbxassetid://115836576619858)
  - Black and green sparkle particles
  - Black and green trail effects
  - Green glow light

## Color Scheme
- Base: Green (#00C800) with hacker/binary patterns
- Effects: Black and green sparkles, trail, and glow
- Glow: Bright green light
